# Resume Descriptions for Epi Predict

You can use these descriptions on your resume depending on the specific role you are applying for. They highlight different aspects of the architecture we built.

### For AI Engineer / GenAI Engineer Roles
**Epi Predict: End-to-End AI Outbreak Forecasting System**
* Designed and deployed a production-grade AI healthcare application forecasting seasonal influenza outbreaks using live global WHO surveillance data.
* Architected a scalable prediction engine utilizing an ensemble of 7 machine learning models (XGBoost, LSTM, Random Forest, ARIMA), achieving high accuracy via automated model selection and hyperparameter tuning.
* Implemented Explainable AI (XAI) using SHAP (SHapley Additive exPlanations) to provide transparent, interpretable feature importance analysis for healthcare professionals.
* Built a complete microservices backend with FastAPI, integrating intelligent modules for risk classification, dynamic health recommendations, and automated outbreak alerts.

### For Data Scientist Roles
**Global Influenza Forecasting & Risk Analysis Platform**
* Developed a comprehensive time-series forecasting pipeline to predict global influenza cases up to 12 weeks in advance using WHO FluNet API data.
* Engineered advanced temporal and lagged features (rolling averages, positivity rates) to capture complex seasonal outbreak dynamics.
* Evaluated multiple algorithmic approaches including deep learning (LSTM), statistical (ARIMA), and gradient boosting (XGBoost), optimizing for R² and RMSE.
* Designed an interactive Streamlit executive dashboard featuring dynamic Plotly visualizations, confidence intervals, and automated reporting.

### For MLOps / Machine Learning Engineer Roles
**Epi Predict: Automated MLOps Healthcare Pipeline**
* Built an end-to-end MLOps pipeline for automated ingestion of WHO epidemiological data, feature engineering, and model training.
* Implemented a custom Model Registry to track versioning, performance metrics (MAE, RMSE, R²), and artifacts for 7 different model architectures.
* Developed a continuous monitoring system logging real-time prediction inputs/outputs to SQLite, alongside system performance tracking.
* Containerized the multi-service application (FastAPI backend + Streamlit UI) using Docker and Docker Compose for seamless cross-environment deployment.

### For Full Stack Python Developer Roles
**Python Healthcare Analytics Application**
* Built a full-stack, data-intensive web application to track and predict infectious disease outbreaks using object-oriented Python architecture.
* Developed a robust REST API using FastAPI with Pydantic schema validation, asynchronous endpoints, and comprehensive error handling.
* Created a highly responsive, premium UI using Streamlit, featuring custom CSS (glassmorphism design), interactive Plotly components, and dynamic routing.
* Orchestrated the application deployment using Docker Compose, integrating data persistence volumes and automated startup scripts via Supervisord.
